# Wordle 워들 블로그 글 생성기

파이썬 설치 없이 브라우저에서 바로 쓰는 네이버 블로그용 Wordle 글 생성기입니다.

## 파일 구성

- `index.html` : 실제 도구 파일입니다. 이 파일 하나만 있어도 실행됩니다.
- `README.md` : 사용 설명서입니다.

## 기능

- 날짜와 정답을 넣으면 제목 자동 생성
- Wordle 번호 자동 계산
- 모음 개수 자동 계산
- 중복 글자 여부 자동 계산
- 첫 글자, 마지막 글자, 단어 모양 자동 생성
- 네이버 블로그용 본문 자동 생성
- 태그 자동 추천
- 이미지 캡션 문구 추천
- 제목/본문/태그 복사 버튼
- TXT 저장 버튼

## GitHub Pages에 올리는 방법

1. GitHub에서 새 저장소를 만듭니다.
2. `index.html` 파일을 저장소 최상단에 업로드합니다.
3. 저장소의 **Settings > Pages**로 이동합니다.
4. **Build and deployment**에서 `Deploy from a branch`를 선택합니다.
5. Branch를 `main`, 폴더를 `/root`로 선택한 뒤 저장합니다.
6. 배포 주소가 생성되면 브라우저에서 바로 사용할 수 있습니다.

## 로컬에서 쓰는 방법

`index.html` 파일을 더블클릭하면 브라우저에서 열립니다.

단, 복사 버튼은 브라우저 보안 설정 때문에 로컬 파일에서는 일부 제한될 수 있습니다.
GitHub Pages에 올리면 정상적으로 동작합니다.

## 사용 팁

- 정답 단어는 제목과 태그에 넣지 않는 편이 스포 방지에 좋습니다.
- 품사와 뜻은 자동 사전 검색 기능이 아니므로 직접 입력해야 합니다.
- 네이버 블로그에서 드래그 힌트를 만들려면 복사 후 해당 글자 색을 배경색과 비슷하게 바꾸면 됩니다.
- 스포방지 이미지는 본문 출력문 사이에 직접 넣으면 됩니다.

## Wordle 번호 자동 계산 기준

이 도구는 `2021-06-19`를 기준으로 날짜 차이를 계산합니다.
예시로 `2026-05-29`는 `#1805`로 계산됩니다.

번호가 다르게 보이면 입력창에서 자동 계산 체크를 해제하고 직접 수정하면 됩니다.
