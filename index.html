Wordle 워들 자동 블로그 글 생성기

NYT Wordle 정답을 가져오고, 네이버 블로그용 힌트·정답·뜻풀이 초안을 만듭니다.

사용 흐름
1) GitHub Actions가 매일 자동으로 data/latest.json과 posts/latest.txt를 만듭니다. 2) 이 화면에서 최신 초안을 불러옵니다. 3) 한국어 뜻/후기만 살짝 고치고 복사해서 네이버 블로그에 붙여넣으면 됩니다.

아직 불러온 데이터가 없습니다.

입력값

자동 분석

제목

이미지 캡션 추천

    본문

    네이버 블로그 태그

    기본값은 정답 단어 태그를 빼는 스포방지형입니다.

    GitHub Actions 수동 실행
    • GitHub 저장소 → Actions → Update Wordle Blog Draft → Run workflow
    • 날짜를 비우면 한국 날짜 기준 오늘, 날짜를 넣으면 해당 날짜 글을 생성합니다.
    • 생성된 글은 posts/latest.txt 또는 이 페이지의 “최신 글 불러오기” 버튼으로 확인하세요.
