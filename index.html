Wordle 워들 블로그 글 생성기

정답, 뜻, 난이도만 넣으면 네이버 블로그용 제목·힌트·정답 공개 문단·태그를 자동으로 만들어줍니다. GitHub Pages에 올리면 파이썬 없이 브라우저에서 바로 사용할 수 있습니다.

입력값

무설치 HTML

사용 팁
  • 정답은 제목이나 태그에 넣지 않는 편이 스포 방지에 좋습니다.
  • 품사와 뜻은 자동 사전 검색이 아니므로 직접 입력하세요.
  • 이미지는 네이버 블로그 편집기에서 스포방지 이미지 위치에 넣으면 됩니다.
  • GitHub Pages에서는 별도 설치 없이 이 파일만으로 실행됩니다.
